• Groups: The two groups compared are Male(1) and Female(0)

• Group variable definition:
  Group variable: WhatIsYourSex
  Definition: This variable classifies students into two groups for analysis, where 1 = Male and 0 = Female.

• Response variable definition:
  Response variable: CurrentCigaretteUse 
  Definition: This variable indicates whether students currently smoke cigarettes within the past 30 days, where 0 = non-smoker and 1 = current smoker.
